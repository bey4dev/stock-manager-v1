Share & Permission

Pastikan spreadsheet bisa diakses oleh aplikasi:

  1. Klik tombol "Share" di pojok kanan atas spreadsheet
  2. Masukkan email Google yang akan digunakan untuk login ke aplikasi
  3. Set permission ke "Editor"
  4. Klik "Send"
‚ö†Ô∏è Penting: Email yang diberi akses harus sama dengan email yang digunakan untuk login ke aplikasi StockManager.
7
Update Konfigurasi

Copy Spreadsheet ID dan update konfigurasi aplikasi:

  1. Copy ID dari URL spreadsheet:
    https://docs.google.com/spreadsheets/d/[COPY_THIS_ID]/edit
  2. Buka file src/config/google-sheets.ts
  3. Update SPREADSHEET_ID dengan ID yang di-copy
  4. Pastikan DEMO_MODE = false
